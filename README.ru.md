# Polls - плагин голосований для Azuriom

Плагин добавляет на сайт раздел "Голосования" (Polls):

- зарегистрированные пользователи видят список опросов и могут проголосовать (один или несколько вариантов - настраивается при создании опроса);
- результаты отображаются в виде прогресс-баров с процентами и количеством голосов;
- администраторы создают опросы, добавляют/редактируют/удаляют варианты ответов, открывают и закрывают голосования, удаляют опросы целиком.

## Установка

1. Скопируйте папку `polls` в директорию `plugins/` вашего сайта Azuriom
   (итоговый путь должен быть `plugins/polls/plugin.json`, `plugins/polls/src/...` и т.д.).
2. Зайдите в админ-панель Azuriom -> "Плагины" и активируйте плагин **Polls**.
   При активации Azuriom автоматически выполнит миграции и создаст таблицы:
   - `polls_polls`
   - `polls_poll_options`
   - `polls_poll_votes`
3. Если миграции не выполнились автоматически, выполните на сервере:
   ```bash
   php artisan migrate
   ```

## Использование

### Для администраторов

- Раздел появится в админке: **Polls** (`/admin/polls`).
- "Создать голосование" - указываете название, описание, минимум 2 варианта ответа,
  и при необходимости включаете "Разрешить выбор нескольких вариантов".
- В списке голосований можно:
  - открыть/закрыть опрос (кнопка с замком) - после закрытия пользователи видят только результаты и не могут голосовать;
  - редактировать - менять название, описание, текст вариантов, добавлять новые варианты, удалять старые (минимум 2 варианта должны оставаться);
  - удалить опрос целиком (вместе со всеми голосами).

### Для пользователей

- Ссылка "Polls" появляется в пользовательском меню сайта автоматически.
- Страница `/polls` - список всех опросов со статусом (Открыт/Закрыт).
- На странице конкретного опроса (`/polls/{id}`):
  - если опрос открыт и пользователь ещё не голосовал - показывается форма с вариантами;
  - после голосования (или если опрос закрыт) - показываются результаты в виде прогресс-баров.
- Голосовать можно только один раз на каждый опрос (повторная попытка вернёт ошибку).

## Права доступа

Плагин регистрирует право `polls.admin` - выдайте его роли модератора/администратора,
чтобы у неё появился доступ к разделу "Polls" в админке.

## Возможные доработки

- Ограничение голосований по ролям (например, только для VIP).
- Дата автоматического закрытия опроса.
- Уведомления о новых опросах.
- Экспорт результатов в CSV.
